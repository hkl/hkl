hkl
===

[![Build Status](https://secure.travis-ci.org/hkl/hkl.png)](http://travis-ci.org/hkl/hkl)


The hkl language

To build right now:
You should only need to do this to update your scripts
`$ aclocal`
`$ autoconf`
`$ automake --add-missing`
`$ automake`

Then as normal:

`$ ./configure`
`$ make`
